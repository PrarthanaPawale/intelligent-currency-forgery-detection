# Intelligent Currency Forgery Detection System

This project detects **Real, Fake, and Invalid currency notes** using
Deep Learning (MobileNetV2) and Image Processing.

## Features
- Currency detection using CNN
- Real / Fake / Invalid classification
- Confidence score
- Streamlit-based UI
- Multilingual support (English & Hindi)
- Admin dashboard with analytics

## Technologies Used
- Python
- Streamlit
- TensorFlow / Keras
- OpenCV
- Pandas
- Plotly

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
